# Wallet76

Wallet76 is an investment portfolio tracker for stocks, ETFs and crypto assets.

It helps users track wallets, transactions, portfolio value, profit/loss, watchlists, alerts and subscriptions in one place.

## Features

- User registration and login
- Email verification
- Password reset
- Stripe subscription system
- Admin subscription bypass
- Multiple wallets
- Buy and sell transactions
- Stocks, ETFs and crypto tracking
- Portfolio dashboard
- 24h top movers
- Watchlists
- Price alerts
- Multi-language interface
- Windows desktop app

## Tech Stack

### Frontend
- React
- CRACO
- Tailwind CSS
- Recharts
- Axios
- Electron

### Backend
- FastAPI
- MongoDB Atlas
- Stripe
- Resend
- yFinance
- CoinGecko

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
- Payments: Stripe
